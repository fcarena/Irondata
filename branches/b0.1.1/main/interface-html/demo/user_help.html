Getting Started

Firstly, remember that if you are not the administrator, you may have limited access to some features of the program.

This will take you to the main reporting screen from where you will be able to choose from the two main functions, Reports , Administration or logout of the system.

The first thing that you will want to do when you login is to change the database that you are working with. All templates and reports are specific to a single database. This means if you can't find the report you are looking for, try changing your current database. To do this click on the 'change' link under the Current Database heading on the side panel. Select the database you require from the dropdown and click Change Database

Reports Function

What can you do?: The versatility of the Golf Report Generator means that the user has a wide range of options within this section. From the left of the screen you can see under the current database, any report templates, previously saved, which are available to your level of access. You will also notice three other headings under the current database. These are the three different types of reports you can view and save. They are:

Trend: This feature allows the user to view the data available over a period or several periods of time. The user can create a new trend report by clicking New Trend Report.

Table: This feature allows the user to view the data available in relation to other data for example people's grade in relation to their college. The user can create a new table report by clicking New Table Report.

List: This feature allows the user to view the data available in the form of listed columns. The user can create a new list report by clicking New List Report.

Pregenerated Reports

When you click on an existing report, the 'saved reports' screen is shown. Each saved report has two options next to it they are View and Delete. Clicking on view will allow the user to view the report on screen or alternatively by clicking the now appeared pdf icon, the user can save the report in a .pdf file format for exporting. By clicking the Delete text, the user can delete a previously generated report.

After either creating a new report or loading a previously saved one you will now notice on the right hand side of the screen, several icons.

Run Report

Clicking the run icon will allow the user to generate a new report from the report template. The user will then be able to view this new report or save it for future reference. The time for generating the report can vary due to the complexity of a given report.

New Templates

Actually clicking the new icon is the same as clicking on the new report option on the left hand side of the screen though it will choose the report type of whichever template you happen to be in at the time. It allows the user to create a report template for generating reports from the available data.

Edit Template

Clicking on the edit icon will enable the user to edit an existing report template. The options are the same as with the new report screen.

Delete

Clicking on the delete icon allows the user to delete a report template as opposed to a specific report. Be certain that this is what you wish to do because this action is irreversible. In accordance clicking on this icon will generate a confirmation window while informing you of the fact that the action is irreversible.

Creating a New Template

Once you've clicked on the new icon, the report generator screen will open. The user now has the option of selecting from the available fields for their report.

Aggregates will display specific values including the maximum and minimum values of a cell, the average, the number of inputs, the number of unique inputs (in the case of simliar entires such as gender) and the percentage of the value in comparrison to the total value of the data available.

An user can also choose to link specific cells of data with other reports with the link to option. This 'Drills' down from one report to another, more specific report. Such as a count of members drilling into a list of members names.

When you have created or edited your report template, simply enter the name of the report in the field next to the save button and then click Save As

Table: A table report is made up of cells constrained by two (optionally three) axis'. For each of the X (horiztonal), Y (vertical) and (optionally) Z axis select the table, column. The user can also set constraints on the data by clicking on the constraint button. By clicking this, the user can restrict the data in the report to the specific areas they are interested in for example; If the user wished to display only people in a certain age group, they would be able to select the people field, the age field and the constraint type of greater or lesser then a specific value in this case the age. When you are happy with the constraints you have chosen, click Done.
